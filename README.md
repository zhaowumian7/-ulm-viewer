# 3D ULM Browser Viewers

Static GitHub Pages bundle for the toXYS 3D ULM browser viewers.

Open `index.html` or deploy this folder as a GitHub Pages site.

Included experiments:

- toXYS 400 Frames - Complex
- toXYS 400 Frames - abs^1/4
- toXYS 400 Frames - SVD
- toXYS Pilot 30 Frames

Large raw/intermediate files are intentionally excluded:

- `*.h5`
- `*.pkl`
- official 98 GiB sample data
